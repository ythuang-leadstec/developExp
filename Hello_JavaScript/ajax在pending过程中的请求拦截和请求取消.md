# ajax在pending过程中选择请求拦截和请求取消
防止重复发送ajax请求


## 请求拦截
+ 用一个数组存储目前处于pending状态的请求
+ 发送请求前先判断这个api请求之前是否已经有还在pending的同类，即是否存在上述数组中
+ 如果存在，则不发送请求
+ 不存在就正常发送并且将该api添加到数组中
+ 请求完结后删除数组中的这个api

## 请求取消
+ 用一个数组存储目前处于pending状态的请求。
+ 发送请求时判断这个api请求之前是否已经有还在pending的同类，即是否存在上述数组中
+ 如果存在，则找到数组中pending状态的请求并取消
+ 不存在就将该api添加到数组中
+ 请求完结后删除数组中的这个api
